# AGV 兩輪差速模擬器：技術功能清單 (V5.1)

本專案是一個高精度、可擴展的多機 AGV 模擬系統，專為驗證複雜倉儲環境下的路徑規劃、動態避障與多機協同演算法而設計。

## 1. 核心架構 (Architecture)
*   **V4.6+ OOP 物件導向重構**：後端採用高度模組化設計。每台 AGV 均為自治實體，封裝了獨立的規劃器、控制器與物理模型。
*   **非同步路徑規劃 (Async Pathfinding)**：A* 演算法在背景執行緒運算，徹底消除規劃時的主程式卡頓，支援多機同時規劃。
*   **V5.1 前端效能引擎**：採用單一繪圖循環與 `useRef` 數據同步技術，徹底解決 `requestAnimationFrame` 造成的記憶體洩漏，維持穩定的 60FPS。
*   **高頻物理積分**：採用 100Hz 內部子步進 (Sub-stepping) 技術，確保在高倍速模擬下物理軌跡依然嚴謹。

## 2. 導航與協同 (Navigation & Collaboration)
*   **物理勢場 A* 演算法**：
    *   **全場域斥力累加**：路徑會根據兩側障礙物的物理壓力，自動尋找力量平衡的「中心谷地」。
    *   **圓弧路徑平滑**：採用「雙重移動平均 (Double Moving Average)」技術，將 A* 的折線自動轉化為符合差速轉向特性的平滑圓弧。
*   **V5.0 多機動態避障**：
    *   **互感應系統**：AGV 之間會實時感知彼此的座標與速度，將其他車輛視為動態圓形障礙物。
    *   **安全護盾 (DWA-Shield)**：局部控制器具備多級防護，發現衝突時會自動執行減速、轉向或停車禮讓。

## 3. 精確控制 (Control)
*   **即時旋轉響應 (Instant Spin Response)**：點擊 Start 的瞬間，如果目標在後方，AGV 會立即開始原地旋轉，實現視覺與邏輯的零延遲。
*   **外擺補償 (Swing-out)**：轉彎時自動向彎道外側偏移 $150\text{mm}$，補償內外輪差，保護內側角落不擦撞。
*   **物理極限鎖定**：嚴格鎖定單輪最高 $3000\text{ RPM}$ ($600\text{mm/s}$)，並具備動態加速度限制。

## 4. 模擬與互動 (Simulation & UI)
*   **動態倍率模擬**：支援 1x、5x, 10x, 15x 模擬加速，並實現前端預測同步。
*   **專注編輯介面**：側邊欄動態顯示選中障礙物資訊，支援座標精確輸入。
*   **自動存檔系統**：佈置場景自動序列化為 `obstacles.json`，支援持久化儲存。
*   **雙重吸附機制**：
    *   **AGV/目標點**：吸附於網格線交點（0, 1000...）。
    *   **障礙物**：吸附於網格格子中心（500, 1500...）。
