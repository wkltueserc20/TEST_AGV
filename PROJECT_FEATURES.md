# AGV 兩輪差速模擬器：技術功能清單 (V5.3)

本專案是一個高精度、具備演算法視覺化能力的多機 AGV 模擬系統。

## 1. 核心架構 (Architecture)
*   **V4.6+ OOP 物件導向重構**：後端高度模組化，每台 AGV 均為自治實體。
*   **非同步路徑規劃 (Async Pathfinding)**：A* 演算法在背景執行緒運算，支援多機同時規劃不卡頓。
*   **V5.1 前端效能引擎**：採用單一繪圖循環與 `useRef` 技術，徹底解決記憶體洩漏，維持 60FPS。
*   **高頻物理積分**：100Hz 內部子步進，確保高倍速模擬下的軌跡精度。

## 2. 導航與視覺化 (Navigation & Visualization)
*   **V5.3 A* 搜尋視覺化**：
    *   **搜尋雲重播**：可視化顯示 A* 探索過的節點（淺藍色半透明區域），展示演算法的「思考過程」。
    *   **逐格流水效果**：搜尋過程以每幀 50 個點的速度動態揭露，呈現像水流一樣的擴散感。
    *   **指紋比對技術**：利用 Search Fingerprinting 解決遙測數據高頻更新導致的畫面閃爍。
*   **物理勢場 A* 演算法**：路徑自動尋找兩牆中線，並透過雙重移動平均生成圓弧曲線。
*   **V5.0 多機協同避障**：實時感知彼此位置，將其他車輛視為動態障礙物。

## 3. 精確控制 (Control)
*   **即時旋轉響應**：點擊 Start 瞬間立即執行轉向，消除起步發呆。
*   **V5.2 起步流暢化**：動態前瞻距離與起步安全寬容度（502mm），解決窄道起步難題。
*   **外擺補償 (Swing-out)**：轉彎時自動向外偏移，補償內外輪差保護車角。
*   **物理極限限制**：嚴格鎖定單輪 3000 RPM，具備加速度限制。

## 4. 模擬與互動 (Simulation & UI)
*   **動態倍率**：支援 1x、5x, 10x, 15x 模擬加速。
*   **專注編輯介面**：側邊欄僅顯示選中障礙物，支援精確座標輸入。
*   **自動吸附機制**：AGV 吸附於線點，障礙物吸附於格中心。
*   **自動存檔**：場景配置自動儲存於 `obstacles.json`。
