# 兩輪差速 AGV 路徑規劃模擬器 - 功能需求說明書

## 1. 專案目標
本專案旨在建立一個直觀、實時的模擬平台，用於測試與驗證「兩輪差速驅動」AGV (無人搬運車) 在複雜環境下的移動路徑規劃與動態避障能力。模擬器需能呈現 AGV 從起始點 A 順利移動至自定義目標點 B 的完整過程，並在遇到突發障礙物時做出合理的繞行反應。

## 2. 核心行走功能
*   **兩輪差速模擬**：精確模擬真實兩輪型機器人的運動特性，透過控制左右輪速差來實現前進、後退與原地轉向。
*   **自動導航**：AGV 具備自動追蹤目標的能力，能計算最優路徑朝向指定的 B 點前進。
*   **動態避障**：在移動過程中，AGV 能即時偵測周遭障礙物，並主動修正路徑以避開碰撞，即使在狹窄空間（如 2 公尺寬的通道）也能做出合理的判斷。
*   **到達判定**：當 AGV 接近目標點 B 的指定範圍內時，系統會自動判定任務完成並停止運動。

## 3. 物理與性能參數
*   **高轉速支援**：最高支援 **3000 RPM** 的馬達轉速，對應實際行走速度約 **600 mm/s**。
*   **大型場域模擬**：提供 **50m x 50m** 的超大型笛卡兒座標地圖，滿足長距離導航測試。
*   **車體規格**：以工業級 AGV 為基準，模擬 **1m x 1m** 的正方形車體足跡。
*   **模擬加速**：支援「加速預覽」功能（如 15 倍速），讓使用者能快速觀察長距離或複雜路徑的最終結果，無需實時等待。

## 4. 互動操作功能
*   **障礙物管理**：
    *   **快速新增**：滑鼠左鍵點擊地圖即可立即放置障礙物。
    *   **形狀切換**：支援「1m x 1m 正方形」與「直徑 1m 圓形」兩種主要障礙物類型。
    *   **精確編輯**：提供側邊欄清單，可直接輸入數值修改障礙物的 X、Y 座標。
    *   **靈活刪除**：可點選特定障礙物後刪除，或一鍵清空場上所有障礙物。
*   **目標設定**：滑鼠右鍵點擊地圖任意位置，可即時變更 AGV 的目的地。
*   **模擬控制**：提供「開始 (Start)」、「暫停 (Pause)」與「重置 (Reset)」功能，隨時掌控模擬流程。

## 5. 數據輸出與視覺化
*   **實時遙測數據**：畫面需同步顯示 AGV 的當前狀態，包括：
    *   當前座標 (X, Y)
    *   當前行進線速度 (V) 與角速度 (ω)
    *   **左輪轉速與右輪轉速 (L/R RPM)**
    *   **左右輪差速數值**
*   **絲滑動畫表現**：模擬過程需保持視覺上的連續性與流暢感，AGV 的移動應為平滑滑行而非跳躍式位移。
*   **視覺輔助**：地圖具備格線系統，並清晰標記目標 B 點位置。
