# AGV 兩輪差速模擬器：技術功能清單 (V5.6)

本專案是一個具備高效能運算、非同步處理與直覺互動能力的多機 AGV 模擬系統。

## 1. 核心架構 (Architecture)
*   **V5.6 非同步併發引擎**：
    *   **背景地圖重算 (Async Costmap)**：障礙物變更後，代價地圖的重算在背景線程執行，不阻塞 WebSocket 通訊與物理模擬。
    *   **地圖更新佇列**：支援快速連續的操作（如快速刪除或 Clear All），系統會自動排隊確保最終數據的一致性。
    *   **啟動解鎖**：修正了啟動時同步計算造成的連線卡死問題，實現秒連秒開。
*   **非同步路徑規劃 (Async Pathfinding)**：A* 搜尋在獨立線程運行，支援多機並行規劃且不影響 UI 流暢度。
*   **V5.1+ 高效能渲染**：徹底根除 `requestAnimationFrame` 記憶體洩漏，維持長時間運行的 60FPS。

## 2. 導航與視覺化 (Navigation & Visualization)
*   **極速 A* 搜尋**：引進預處理代價地圖技術，將搜尋複雜度從 $O(N \times M)$ 降為 $O(N)$，規劃速度提升 100 倍。
*   **搜尋過程視覺化**：具備「思考雲」重播功能，以逐格流水動畫展示 A* 演算法的探索行為。
*   **物理勢場平衡**：路徑自動尋找幾何中線，搭配雙重移動平均生成圓弧曲線。

## 3. 互動與操作 (Interaction)
*   **雙擊快速刪除**：在畫布上直接「雙擊」障礙物即可秒速刪除，無需透過選單。
*   **強大 ID 系統**：採用時間戳記 + 隨機碼產生的唯一 ID，具備 Legacy 障礙物 ID 自動補全功能，保證刪除操作 100% 成功。
*   **智慧目標設定**：右鍵點擊設定目標時，若未選取車輛，系統會自動選取預設車輛操作，減少操作步驟。
*   **起步流暢化**：動態前瞻距離與安全寬容度，徹底解決窄道起步死鎖。

## 4. 物理與模擬 (Physics)
*   **外擺補償 (Swing-out)**：轉彎時自動向外偏移補償內外輪差。
*   **物理極限鎖定**：嚴格限制單輪最高 3000 RPM，具備加速度限制。
*   **動態倍率**：支援 1x、5x, 10x, 15x 模擬加速。
